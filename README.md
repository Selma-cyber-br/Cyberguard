# CyberGuard - Cybersecurity Agency Landing Page

## Overview

CyberGuard is a modern cybersecurity agency landing page designed to showcase cybersecurity services, increase security awareness, and provide businesses with information about protecting their digital assets.

The project was created to practice frontend development, responsive web design, and professional cybersecurity branding while simulating a real-world cybersecurity company's online presence.

---

## Features

* Modern cybersecurity-themed design
* Responsive layout for desktop and mobile devices
* Professional landing page structure
* Services section
* About section
* Contact section
* Smooth navigation experience
* Clean and user-friendly interface

---

## Services Showcased

### Security Awareness Training

Helping organizations educate employees about cyber threats, phishing attacks, and security best practices.

### Network Security Consulting

Providing guidance on securing network infrastructures and reducing security risks.

### Risk Assessment

Identifying vulnerabilities and evaluating potential cybersecurity threats.

### Phishing Awareness

Training users to recognize and respond to phishing attempts and social engineering attacks.
